Student Management System (C++)

A console-based Student Management System built in C++ that allows users to manage student records, including adding, editing, searching, displaying, and storing student information in a file.

Features
Add Student – Create new student records with ID, name, and marks.
Edit Student – Update existing student details.
Search Student – Find a student by ID and display their information.
Show All Students – Display all students in a neatly formatted table.
Save to File – Persist student data to a file (students.txt).
Load from File – Load student data from a file.
Input Validation – Ensures correct data types and valid input ranges.
User-friendly Menu – Colored and organized console menu for easy navigation.
Installation
Ensure you have a C++ compiler (e.g., g++) installed on your system.
Clone or download this repository.
Make sure student.hpp is included in the same directory as main.cpp.
Compile the program:
g++ -std=c++17 -o student_management main.cpp
Run the program:
./student_management
